# Sons d'Abyssus

Depose ici tes propres fichiers audio (format `.mp3`). Ils remplacent
automatiquement les sons synthetises par defaut. Noms attendus :

| Fichier               | Declenchement                                  |
|-----------------------|------------------------------------------------|
| `quest-accept.mp3`    | Clic pour accepter une quete / un evenement    |
| `quest-refuse.mp3`    | Clic pour refuser une quete                    |
| `answer-correct.mp3`  | Bonne reponse a une question                   |
| `answer-wrong.mp3`    | Mauvaise reponse a une question                |
| `boss-defeat.mp3`     | Defaite contre un boss (HP a 0 pendant un boss)|

Conseils : garde des fichiers courts (< 2 s, sauf la defaite boss), volume
homogene. Tant qu'un fichier est absent, un son synthetise est joue a la place.
